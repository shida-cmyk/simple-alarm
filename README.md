# Simple Alarm ⏰

シンプルで軽量な、ブラウザだけで動くアラームアプリです。
フレームワークもビルドも不要。`index.html` を開くだけで使えます。

## 特長

- 🕐 リアルタイム時計表示
- ➕ 時刻を選んでワンタップでアラームをセット
- 🔔 Web Audio API によるビープ音（音声ファイル不要）
- 💾 localStorage でアラームを保存（リロードしても消えない）
- 📱 スマホ・PC 両対応のレスポンシブなダークUI
- 🪶 依存ゼロ・HTML/CSS/JS のみ

## 使い方

1. `index.html` をブラウザで開く
2. 時刻を選んで「セット」を押す
3. 設定した時刻になるとオーバーレイとビープ音で通知
4. 「停止」を押して解除

> ※ ブラウザのタブが開いている間に動作します（バックグラウンドのタブでは音声が抑制される場合があります）。
